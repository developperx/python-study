# Python3エンジニア認定実践試験 学習アプリ 🐍

「**Python3エンジニア認定実践試験**」の合格を目指すための、**無料・ブラウザで動く**学習アプリです。
模擬試験・章別演習・弱点復習・進捗管理を備え、全 **500問**（オリジナル作問）を収録しています。
各問は正解の解説に加え、**誤答を含む全選択肢に「なぜ正解／不正解か」の個別理由**を表示します。

> ⚠️ **非公式アプリです。** 本アプリは一般社団法人Pythonエンジニア育成推進協会／株式会社オデッセイ コミュニケーションズとは一切関係ありません。
> 掲載問題はすべてオリジナル作問で、**公式試験問題や教材『Python実践レシピ』の文章は転載していません**。

## 公開URL（Vercel）

`main` へ push すると Vercel が自動でビルド・検証・デプロイします。PR を作成するとプレビュー URL も自動生成されます。

- 公開URL: **https://python-study.vercel.app/**
- 初回のみ [Vercel Dashboard](https://vercel.com/) で "Add New Project" → GitHub リポジトリ `developperx/python-study` をインポートしてください（`vercel.json` の設定が自動で読み込まれます）。

## 主な機能

| モード | 内容 |
|--------|------|
| 📝 模擬試験 | 本番同様 **40問 / 制限時間75分 / 合格ライン70%**。出題範囲の比率に沿って各章から出題し、合否判定と章別の正誤を表示。 |
| 📚 章別演習 | 出題対象の15章ごとに集中ドリル。即時採点＋解説。 |
| 🎲 ランダム練習 | 章・難易度・出題数を指定してランダム出題。 |
| 🔁 弱点復習 | 間違えた問題・ブックマークした問題だけを再挑戦。 |
| 📊 成績 | 章別正答率、模試の履歴、総合正答率を可視化。 |

- 全問に**総合解説＋選択肢ごとの正誤理由＋学習リファレンス**付き
- 進捗・成績は**ブラウザの localStorage** にのみ保存（サーバ・アカウント不要）
- ダークモード対応・スマホ対応（レスポンシブ）

## ローカルで動かす

`fetch` で JSON を読み込むため、`file://` では動作しません。簡易Webサーバ経由で開いてください。

```bash
python3 -m http.server 8000
# ブラウザで http://localhost:8000/ を開く
```

## 構成

```
index.html              SPA本体
css/style.css           スタイル（レスポンシブ／ダークモード）
js/
  app.js                ルーティング・画面描画・クイズ進行・タイマー
  data.js               問題データの読込／試験構成(EXAM)
  quiz.js               出題・採点ロジック（模試/章別/ランダム/復習）
  storage.js            localStorage 永続化（進捗・成績・ブックマーク）
  ui.js                 描画ヘルパー
data/
  index.json            章メタ
  chNN.json             章別問題データ（tools が生成）
tools/
  define_questions.py   問題の定義（オリジナル作問）
  build_questions.py    JSON生成＋コード出力問題の自動検証
  validate_data.py      生成データの整合性チェック
docs/CONTRIBUTING-QUESTIONS.md  作問ガイド
```

## 問題の追加・編集

問題は `tools/define_questions.py` に Python で定義し、ビルドして JSON を生成します。
`code` を含む出力予測問題に `verify=True` を付けると、**実際にコードを実行して正解と一致するか自動検証**されます。

```bash
python3 tools/build_questions.py   # data/ を再生成＋コード検証
python3 tools/validate_data.py     # id一意性・answer範囲・必須項目・模試比率を検証
```

詳しくは [docs/CONTRIBUTING-QUESTIONS.md](docs/CONTRIBUTING-QUESTIONS.md) を参照してください。
